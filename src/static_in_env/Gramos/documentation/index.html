1 . Getting Started

This documentation allows you to quickly build your projects with Gramos Admin. After downloading the files you will see a screen as follows.

2 . Setup

2.1 Starting with Gulp

With Gulp, you can quickly handle frontend operations, compress your files, or merge them faster with SCSS.

2.2 Gulp Installation

You need to install Node.js before you start using Gulp. Node.js with NPM (Node Packaged Modules) you can manage Gulp. You can manage gulp with NPM (node packaged modules) in Node.js. Follow these steps to setup.

  1. Open the Get Started folder in the file you downloaded.
  2. Arrange the Gulp configuration file (Gulpfile.js) for you.
  3. Go to the Get Started folder on the console screen and run the npm install command
  4. Then run the gulp command 

Gulp keeps track of CSS and Javascript files, automatically detects changes and applies the rules in the configuration file.

You can now start developing your Gramos Admin theme.
